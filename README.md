# WindOps BC Analyzer

风电运维商业案例智能分析工具 — 基于本地 AI 的成本推断与多方案对比系统。

## 版本

| 分支 | 版本 | 说明 |
|------|------|------|
| `main` | v1.1 | 稳定发布版 |
| `develop` | **v1.2-dev** | 开发版：5方案 + 综合评分 + 可视化仪表盘 + 权重配置 |

## v1.2-dev 新增内容

### 可视化仪表盘

| 图表 | 说明 |
|------|------|
| 📊 堆叠柱状图 | 5 方案 × 成本构成并排对比 |
| 🎯 雷达图 | 各方案在 4 个评分因子上的表现 |
| 📍 散点图 | 完成时间 vs 总成本，识别 Pareto 最优 |
| 🍩 环形图 | 选中方案的成本构成占比 |

AI 推荐结论卡片：推荐理由 + 风险提示 + 推理过程。

### 综合评分权重配置

在 AI 配置页新增「评分权重」Tab，可实时调整 4 个因子权重和罚款风险缓冲阈值。权重持久化到 DB，规则引擎实时读取。

### 结果导出

| 格式 | 说明 |
|------|------|
| CSV | ✅ 5 方案对比表 |
| Excel | 🆕 含计算明细的工作表 |
| PDF | 🆕 浏览器打印 PDF |

## 规则引擎（元数据驱动）

46 条预置规则按 condition_json 自动匹配方案参数：

| 维度 | 规则数 | 说明 |
|------|--------|------|
| 零部件成本 | 5 | 丹麦基准×1.0、中国×0.70、发达溢价×1.05、发展中折价×0.90、NC溢价×1.30 |
| 管理费 | 3 | 丹麦10%、中国3%、本地0% |
| 运输费用 | 13 | 按来源×目的国×运输方式（空运=海运×6.5） |
| 运输天数 | 7 | 基础天数 + 台风季Buffer |
| 罚款权重 | 10 | 11国时效敏感度（0.70~1.30） |
| 历史数据处理 | 6 | 中位数聚合、异常检测、等权处理 |
